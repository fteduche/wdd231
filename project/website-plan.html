Discover Abuja: A Tourist Guide

Project Overview

This website project aims to create a comprehensive tourist guide for Abuja, Nigeria's capital city. The site will feature various sections showcasing Abuja's rich cultural heritage, historical landmarks, natural attractions, and modern amenities.

Site Purpose

The purpose of this website is to provide a valuable resource for tourists planning to visit Abuja. The site will offer information on popular attractions, outdoor activities, restaurants, hotels, and shopping centers, as well as a calendar of events and festivals.

Scenarios

  • What are the top tourist attractions to visit in Abuja?

    This question will help drive content related to popular landmarks, cultural spots, and must-see attractions in Abuja, including descriptions, images, and recommendations based on visitor preferences.

  • What are the best outdoor activities to enjoy in Abuja?

    This scenario will guide content about Abuja's natural attractions, outdoor activities such as hiking and exploring the surrounding savannas, and recommendations for adventure seekers and nature enthusiasts.

Color Schema

The color schema for this site includes two main colors:

  • Caribbean Current (hsla(182, 100%, 23%, 1)): This color is used for headings and accents, giving the site a vibrant and tropical feel.
  • Floral White (hsla(47, 43%, 96%, 1)): This light, neutral color is used as the background color to provide a clean and pleasant visual experience.

Example CSS HSL:

    /* CSS HSL */
    --black: hsla(0, 0%, 0%, 1);

Typography:

1. Primary Font: Oswald (sans-serif)

  • Usage: Oswald will be used for all headings across the site to give a bold, modern look and help guide users as they explore different sections.
  • Applied to:
    • Headings (h1, h2, h3, h4): All headings on the website will use Oswald to clearly define sections and make it easy for users to navigate.

2. Secondary Font: Source Sans Pro (sans-serif)

  • Usage: Source Sans Pro will be used for all body text and paragraphs, ensuring that the content is easy to read and visually appealing for users.
  • Applied to:
    • Body Text (p): All paragraphs, descriptions of tourist attractions, and general content on the site will use Source Sans Pro for readability.
    • Links and Buttons: Interactive elements like links and buttons will also use Source Sans Pro to maintain consistency in design.
